# Cat Land Like its 1999

This is a site that I made in a Windows 98 emulator. As a Mac user, I found it interesting how I was forced to use control instead of the obviously superior command (for legal purposes, this is a joke).

I really enjoyed messing around with old HTML features that aren't around anymore such as marquee and bgsound. I also learned how to use tables instead of flexbox for formatting. 

While it is a very simple site, I really enjoyed making it! And hey, maybe I'll improve it sometime down the road and make it a proper 90s site
